📊 Customer Purchase Behavior Analysis

Data Analytics & Data Profiling Project


---

🎯 Project Overview

This project analyzes Customer Purchase Behavior using Python. The dataset is collected from CSV, JSON, SQL Database, and API sources. The data is then cleaned, analyzed, and visualized to discover meaningful insights that help understand customer purchasing patterns and business trends.


---

🎯 Objectives

✅ Import data from different sources (CSV, JSON, SQL, API)

✅ Clean and preprocess the dataset

✅ Handle missing values and duplicate records

✅ Perform Exploratory Data Analysis (EDA)

✅ Analyze relationships between customer attributes

✅ Generate an automated Data Profiling Report
